# Projeto Informativo: Jair Messias Bolsonaro 🇧🇷

Este é um projeto de desenvolvimento web focado na criação de uma página informativa, biográfica e multimídia sobre Jair Messias Bolsonaro, 38º Presidente do Brasil. O site foi construído utilizando tecnologias fundamentais da web (HTML5, CSS3 e JavaScript) com foco em design responsivo, acessibilidade e interatividade.

---

## 🎯 Objetivo do Projeto

O objetivo principal deste projeto é consolidar conhecimentos práticos de desenvolvimento front-end. O site serve como um repositório dinâmico de informações, estruturando dados biográficos, marcos históricos da carreira política e registros multimídia (como a incorporação de vídeos do YouTube) em uma interface limpa, moderna e de fácil navegação.

---

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando a tríade padrão do desenvolvimento web:

- **HTML5:** Estruturação semântica do conteúdo (uso de tags como `<header>`, `<main>`, `<section>`, `<table>` e `<iframe>` para o vídeo do YouTube).
- **CSS3:** Estilização visual completa, incluindo layout responsivo (adaptável para celulares, tablets e computadores), variáveis de cores (tons de azul, verde e cinza) e estilização de elementos como tabelas e contêineres de vídeo.
- **JavaScript (Vanilla):** Implementação de interatividade e lógica do lado do cliente (pode ser usado para menus responsivos, alternância de tema escuro/claro ou manipulação de elementos da página).

---

## ⚙️ Funcionalidades

- **Design Totalmente Responsivo:** O site se adapta a qualquer tamanho de tela, garantindo uma boa experiência tanto em computadores quanto em dispositivos móveis.
- **Vídeo Incorporado (Responsivo):** Inclusão de registros em vídeo através da API do YouTube, configurado para manter a proporção de tela 16:9 de forma automática.
- **Tabela Informativa:** Apresentação clara e organizada dos dados biográficos cruciais (data de nascimento, naturalidade, cargos ocupados).
- **Semântica e Acessibilidade:** Uso correto de tags estruturais para melhorar o ranqueamento em motores de busca (SEO) e leitura por leitores de tela.
